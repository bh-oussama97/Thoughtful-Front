Overview

ThoughtFul is a basic application for a blogging platform, designed using .NET 6 WEB API style. It's clean, modern, and built for developers who value simplicity and vertical feature organization.

Architecture Highlights

  • Vertical slices architecture (feature-based organization)
  • CQRS pattern using MediatR
  • Auto-registered modules per feature
  • DTO mapping with AutoMapper
  • Rich domain model (to a degree)
